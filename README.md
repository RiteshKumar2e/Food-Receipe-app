# Gourmet Haven - Food Recipe App

A professional food recipe application built with React.js and Node.js.

## Structure
- `frontend/`: React + Vite frontend with modern glassmorphism design.
- `backend/`: Node.js + Express backend for API integration.

## Getting Started

### 1. Backend
```bash
cd backend
npm install
npm start
```

### 2. Frontend
```bash
cd frontend
npm install
npm run dev
```

## Features
- Search for recipes by name.
- Professional, responsive UI with Glassmorphism.
- Detailed recipe views with ingredients and instructions.
- Integration with TheMealDB API.
